Breve apresentação da história dos Computadores

(Voltar)

Nos primeiros módulos é introduzido uma breve história dos primeiros computadores e história da WEB.

A primeira geração de computadores foi criada em 1944 chamada de Harvard Mark I, dois após em 1946 foi criado o Colossus e nos mesmo ano o ENIAC.

A segunda geração foi de 1959 à 1965, onde fomos apresentados aos transistores o que consequentemente reduziria o tamanho dos computadores drasticamente.

A tercerita geração foi de 1965 à 1970, onde o circuito integrado foi apresentado, diminuindo ainda mais seus tamanhos.

A quarta geração a partir de 1971 já é o que conhecemos como computadores atualmente, há quem diga que já estmos na quinta geração, que seria as inteligências artificiais, computadores quanticos, bitcoins e metaverso.

Entendendo Comunicação Client x Server

(Voltar)

Servidores são locais onde são hospedados os sites em forma de que um cliente (computador) possa ter acesso através de um link pela internet, sem o site estar hospedado em uma espécie de servidor fica impossibilitado obter acesso a ele.

Clientes (Clients) são os computadores que utilizamos para obter acesso a sites que desejamos navegar, são enviadas as informações para internet por meio do protocolo HTTP que irá bucar no servidor caso esteja hospedado.

Sobre Navegadores

(Voltar)

Navegadores, também chamados de browses são programas criados por empresa utilizados de forma gratuita para navegação na internet.

Iniciou-se com o MOSAIC, depois passou para o Netscape e nos dias atuais há vários navegadores.

O que são Aplicações Web

(Voltar)

São soluções que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a instalação de aplicativos.

Um exemplo é o famoso site de conteúdos de vídeos YouTube, onde podemos ascessa-lo por meio do site www.youtube.com.br, sendo assim o YouTube é considerado uma aplicação web e também uma aplicação não web já que podemos baixar seu aplicativo em um dispositivo móvel, como um SmartPhone.

Breve Introdução ao HTML

(Voltar)

Hypertext Markup Language (HTML) não é uma linguagem de programação e sim uma linguagem de marcação como seu próprio nome sugere.

A estrutura básica de um documento HTML consiste em elementos essenciais, como o !DOCTYPE, html, head, e body essas são chamadas tags, toda tag deve-se iniciar com "< >", formando assim a estrutura básica do HTML. Tags HTML são elementos usados para marcar e formatar o conteúdo HTML.
